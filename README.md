
# Squares 3

By BranacleBoy

> An HTML5 game heavily inspired by Squares 2 by Gavin Shapiro


## Controls

Use your mouse to move your square around to collect black squares while avoiding the red ones.


## Built With

HTML5
p5.js - https://p5js.org/
